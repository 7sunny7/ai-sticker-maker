# AI Sticker Maker

AI Sticker Maker 是一个面向创作者的 AI 表情包网格大图生成工具。

第一版目标是让用户选择 8 / 16 / 24 张，输入主题或抽方向，系统自动生成表情条目，并生成一张带中文文字的表情包网格大图。

## MVP 定位

- 面向微信表情投稿创作者、轻 IP 创作者和账号运营者。
- 第一版只验证含字网格大图生成链路。
- 第一版先做 Web App，不先做微信小程序。
- 第一版不做积分、上架、切图、单张补图和投稿包导出。
