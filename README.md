# AI Sticker Maker

AI Sticker Maker 是一个面向创作者的 AI 微信表情包制作工具。

第一版目标是通过文字生成原创角色，再基于角色卡分张生成 8 张统一风格的静态表情图，并导出基础微信投稿素材包。

## MVP 定位

- 面向微信表情投稿创作者、轻 IP 创作者和账号运营者。
- 第一版只做文字生成原创角色表情包。
- 第一版先做 Web App，不先做微信小程序。
- 不做自动上架微信表情商店，用户仍需手动去微信表情开放平台投稿。
